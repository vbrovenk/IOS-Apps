# WeatherShow App

Application shows current weather with animation on top of screen, hourly weather in horizontal stack and weather for week of chosen location.

### Key objectives

- SwiftUI
- MVVM
- Network API calls
- Lottie Animation
- work with JSON
- Date and Time formatters
- Observable and Observed objects
